# 傳入、即時資料傳輸的技術規範{#technical-specifications-for-inbound-real-time-data-transfers}

第三方內容提供商可以期望根據這些技術規範與Audience Manager交換資料。 即時（同步）整合在用戶訪問或在您的站點上執行操作時幾乎即時地傳輸資料。 技術、工程或開發團隊應使用此材料幫助設定即時資料傳輸，並提供Audience Manager。
