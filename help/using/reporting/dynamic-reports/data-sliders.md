# 使用資料滑桿篩選報表結果{#filter-report-results-with-the-data-sliders}

使用各種報表滑桿，只顯示落在您指定範圍內、上方或下方的資料。

<!--

 -->

**使用資料滑桿設定報表結果的下／上範圍**

報表滑桿可讓您設定互動報表傳回資料的限制。 移動左滑桿，排除特定值以下的資料。 向右移動滑桿以排除特定值以上的資料。 報表會更新並傳回落在所需範圍內的資料。 使用滑桿可：

* 減少報表傳回的資料總量。
* 專注在特定大小範圍內的特徵或群體。
