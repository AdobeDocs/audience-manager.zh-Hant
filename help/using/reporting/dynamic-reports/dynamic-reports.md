# 互動式和重疊報表{#interactive-and-overlap-reports}

互動式報表會顯示特徵和區段的效能和重疊資料。 這些報表不會使用以欄和列排列的數字，而是會使用不同的形狀、顏色和大小傳回資料。 此外，您可以選擇個別或資料點群組，並深入研究報告結果以取得更多詳細資訊。 這些視覺化技巧和報表互動有助於讓大量數值資料更易於理解。
