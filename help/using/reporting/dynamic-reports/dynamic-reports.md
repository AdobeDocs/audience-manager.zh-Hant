# 互動式和重疊報表{#interactive-and-overlap-reports}

互動式報告顯示特徵和段的效能和重疊資料。 這些報告不使用排列在列和行中的數字，而是使用不同的形狀、顏色和大小返回資料。 此外，您可以選擇單個或一組資料點，並細化到報表結果，以瞭解更多詳細資訊。 這些可視化技術和報告交互性有助於使大量數字資料更易於理解。
