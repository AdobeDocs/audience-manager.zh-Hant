# Interactive and Overlap Reports{#interactive-and-overlap-reports}

互動報表會顯示特徵和區段的效能和重疊資料。這些報告不會使用以欄和列排列的數字，而是使用不同的形狀、顏色和大小傳回資料。此外，您可以選擇個別或資料點群組，並深入探究報表結果，以瞭解詳細資訊。這些視覺化技巧和報表互動功能可讓您更容易瞭解大量的數值資料。
