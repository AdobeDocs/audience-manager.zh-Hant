# 回報的轉換特徵{#reported-conversion-traits}

「轉換性狀」報告顯示某個日期標籤為轉換組的所有性狀。

轉換組的轉換特徵可以從報告運行更改為報告運行。 此報表按所選報告日期的轉換組顯示轉換特徵。
