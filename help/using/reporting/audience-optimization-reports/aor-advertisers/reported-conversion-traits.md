# 回報的轉換特徵{#reported-conversion-traits}

「轉換特徵」報表會針對特定日期的轉換群組顯示所有特徵，並標示為轉換特徵。

轉換群組的轉換特徵可能會從報表執行變更為報表執行。 報表會依轉換群組顯示所選報表日期的轉換特徵。
