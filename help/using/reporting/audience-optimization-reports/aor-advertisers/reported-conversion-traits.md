# 回報的轉換特徵{#reported-conversion-traits}

「轉換特徵」報表顯示所有在特定日期標示為轉換群組轉換特徵的特徵。

轉換群組的轉換特性可從報告執行變更為報告執行。 報表會依所選報表日期的轉換群組顯示轉換特徵。
