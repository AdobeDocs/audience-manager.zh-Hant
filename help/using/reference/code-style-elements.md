# 程式碼和文字元素的樣式慣例{#style-conventions-for-code-and-text-elements}

這些元素會識別說明檔案中使用的程式碼選項和變數。 通常，您不會將這些符號或樣式元素包含在程式碼或資料檔案中。 它們只是視覺指示器。

  </tr>

   <td colname="col1"> <p> <code> [optional]</code> </p> </td>
