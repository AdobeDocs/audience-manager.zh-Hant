# 程式碼和文字元素的樣式慣例{#style-conventions-for-code-and-text-elements}

這些元素會識別說明檔案中使用的程式碼選項和變數。 通常，您不會在程式碼或資料檔案中加入這些符號或樣式元素。 它們只是視覺指標。

  </tr>
