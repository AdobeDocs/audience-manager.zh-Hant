# 程式碼和文字元素的樣式慣例{#style-conventions-for-code-and-text-elements}

這些元素標識了幫助文檔中使用的代碼選項和變數。 通常，您不會在代碼或資料檔案中包含這些符號或樣式元素。 它們只是視覺指示器。

  </tr>
