# 按篩選條件搜尋訊號 {#search-signals-by-filters}

將鍵值欄位保留為空白，以搜尋更廣範圍的訊號，並使用可用的濾鏡來縮小結果。

當您未考慮特定的鍵值對，但想要查看特定時段內多個信號的演化時，請使用此方法。

在下例中，這些濾波器被配置為顯示過去30天中所有未使用的信號，最少計數為1000。
