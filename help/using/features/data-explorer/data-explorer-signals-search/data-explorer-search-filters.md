# 按篩選條件搜尋訊號 {#search-signals-by-filters}

將鍵值欄位留空以搜索範圍更廣的信號，並使用可用的過濾器縮小結果範圍。

當您沒有特定的鍵值對，但希望查看多個信號在特定時間段內的演化時，請使用此方法。

在下面的示例中，過濾器配置為顯示過去30天中所有未使用的信號，最少計數為1000。
