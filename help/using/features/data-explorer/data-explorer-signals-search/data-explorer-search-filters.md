# Search Signals by Filters {#search-signals-by-filters}

將索引鍵值欄位留空，以搜尋更廣泛的訊號，並使用可用的篩選器縮小結果。

當您不考慮特定機碼值對時，請使用此方法，但想要查看特定時段內多個訊號的演化。

在下列範例中，篩選器設定為顯示過去30天內所有未使用的訊號，最低計數為1000。
