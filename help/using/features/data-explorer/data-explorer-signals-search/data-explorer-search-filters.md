# 依篩選條件搜尋訊號 {#search-signals-by-filters}

將索引鍵值欄位保留空白可搜尋較廣泛的訊號，並使用可用的篩選器來縮小結果範圍。

如果您沒有特定的機碼值組，但想要檢視多個訊號在特定時段內的演變，請使用此方法。

在以下範例中，篩選器已設定為顯示過去30天所有未使用的訊號，最小計數為1000。
