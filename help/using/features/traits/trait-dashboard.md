# 特徵控制面板 {#traits-dashboard}

特徵控制面板是用於管理特徵的集中工作區。 您可以導覽至&#x200B;**[!UICONTROL Audience Data]** > **[!UICONTROL Traits]** ，查看[!UICONTROL Traits]控制面板。

![](assets/traits-dashboard.png)

<!-- c_tb_dashboard.xml -->

[!UICONTROL Traits]控制面板包含可協助您：

1. 在表格中查看您所有的特徵和相關詳細資訊，其中包含您可排序的欄。
2. 檢閱並使用[作用中受眾特徵和資料來源同步特徵](../../features/traits/client-activity-synced-audience-traits.md)。
3. 建立、編輯和刪除特徵。
4. 檢視及管理特徵儲存資料夾。
5. 依名稱、ID、說明或資料來源搜尋特徵。 在搜索時按一下資料夾以將結果限制為該資料夾及其子資料夾。
6. 依特徵類型（已上線、規則型、演算法、資料夾特徵）篩選特徵。
