# 我們的唯讀使用者為何能夠建立、編輯或刪除特徵和區段？

**問題**

我們的唯讀使用者為何能夠建立、編輯或刪除特徵和區段？

**回答**

除了在管理區段中建立群組和權限之外，您還需要聯絡客戶服務(amsupport@adobe.com)，以便代表為您的帳戶啟用角色存取控制(RBAC)。
